Self-Botting & Automated Farming

ZERO TOLERANCE

Violation of this section results in an immediate permanent blacklist.

"Self-botting" refers to the use of a user account (not a bot account) to send automated messages or commands, or the use of any script, macro, program, or tool to automate interactions with Kita on your behalf.

"Automated farming" refers to any method of triggering Kita's reward systems — including but not limited to daily pulls, coin collection, or activity bonuses — without genuine manual user interaction.

Examples of prohibited automation include, but are not limited to:

  • Using a selfbot, auto-typer, keyboard macro, or similar tool to send commands automatically
  • Writing or using scripts that interact with Discord's API using a user token to trigger Kita
